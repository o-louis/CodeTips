There are multiple methods to check if an array contains a value :

<Code>{
`const sports = ["soccer", "hockey", "basketball", "baseball", "ski"];\n
//1. with indexOf
sports.indexOf("hockey") !== -1 // true\n
// 2. with includes
sports.includes("hockey"); // true\n
// 3. with some
sports.some(sport => sport === 'hockey'); // true\n`
}</Code>
