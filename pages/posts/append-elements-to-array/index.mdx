1. Add element to the beginning - **unshift()**
<Code>{
`const countries = ["USA", "Japan", "France", "England"];
countries.unshift("Australia", "China");\n
// ➔ ["Australia", "China", "USA", "Japan", "France", "England"]`
}</Code>

2. Add element to the end<br /><br />
  **mutate the original array**
<Code>{
`const countries = ["USA", "Japan", "France", "England"];\n
// 1. with push
countries.push("Italy", "Russia");\n
// 2. with spread features
const coolCountries = ["Italy", "Russia"];
//
const countries = "hey";
// 3. with splice

// ➔ ["USA", "Japan", "France", "England", "Italy", "Russia"]`
}</Code>

  **non-mutate the original array and create a new one**
<Code>{
`const countries = ["USA", "Japan", "France", "England"];\n
// 1. spread
const arr = [...countries, "Italy", "Russia"];\n
// 2. concat
const arr = countries.concat("Italy", "Russia");\n
// ➔ arr = ["USA", "Japan", "France", "England", "Italy", "Russia"]`
}</Code>
