There are two ways to extract part of a string.

**substr()** begins at the character specified position and returns the specified number of characters.
**substring()** returns the part of the string between the start and end indexes, or to the end of the string

<SyntaxHighlighter
  language="javascript"
  style={shadesOfPurple}
  className="syntaxhighlighter font-sans rounded-md"
>
  {`const str = "Hello World !"\n
// 1. substr(start, length)
const word = str.substr(6, 5);\n
// 2. substring(start, end)
const word = str.substring(6, 11);\n
// -> word = "World"`}
</SyntaxHighlighter>
