1. Add to the beginning - [unshift](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/unshift)

   <Code>{`const countries = ["USA", "Japan", "France", "England"];
countries.unshift("Australia", "China");\n
console.log(countries);
// expected output: ["Australia", "China", "USA", "Japan", "France", "England"]`}</Code>

2. Add to the end<br /><br />
   **mutate the original array**

   <Code>{`const countries = ["USA", "Japan", "France", "England"];\n
// 1. with push
countries.push("Italy", "Russia");\n
// 2. with spread operator
const coolCountries = ["Italy", "Russia"];
countries.push(...coolCounties);\n
// 3. with splice
countries.splice(countries.length, 0, 'Italy', 'Russia');\n
console.log(countries);
// expected output: ["USA", "Japan", "France", "England", "Italy", "Russia"]`}</Code>

   **non-mutate the original array and create a new one**

   <Code>{`const countries = ["USA", "Japan", "France", "England"];\n
// 1. with spread operator
const arr = [...countries, "Italy", "Russia"];\n
// 2. with concat
const arr = countries.concat("Italy", "Russia");\n
console.log(countries); // ["USA", "Japan", "France", "England"]
console.log(arr); // ["USA", "Japan, "France", "England", "Italy", "Russia"]`}</Code>
