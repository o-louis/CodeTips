It exists different methods to removing element from an array.

1. Remove the first element - **shift()**
<Code>
  {`const things = ["knife", "bottle", "keyboard", "table", "tv"];
things.shift();\n
// ➔ ["bottle", "keyboard", "table", "tv"]`}
</Code>

2. Remove the last element - **pop()**
<Code>
  {`const things = ["knife", "bottle", "keyboard", "table", "tv"];
things.pop();\n
// ➔ ["knife", "bottle", "keyboard", "table"]`}
</Code>

3. Remove from a specific index - **splice()**
<Code>
  {`const arr = ["toto", "tata", "tutu", "tete"];
const index = 2;
arr.splice(index, 1);\n
// ➔ ["toto", "tata", "tete"]`}
</Code>

4. Remove element by value - **filter()**
<Code>
  {`const fruits = ["apple", "orange", "pear", "banana", "coconut", "peach"];
const fruitToRemove = "pear";
const filtered = fruits.filter(item => item !== fruitToRemove);\n
// ➔ filtered = ["apple", "orange", "banana", "coconut", "peach"]`}
</Code>
