We use [find()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find) which is a function that returns the first value in the array that passes the provided testing function.

<Code>{`const users = [
  { id: 1, name: 'Pedro' },
  { id: 2, name: 'Clara' },
  { id: 3, name: 'Rodriguez' }
];\n
users.find(user => user.id === 2); // { id: 2, name: 'Clara' }`}</Code>
