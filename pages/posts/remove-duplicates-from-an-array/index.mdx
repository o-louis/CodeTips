To extract the unique values from array, there are two solutions using the ["Set"](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Set) object. It's a data structure that lets you store unique values of any type.

<SyntaxHighlighter
  language="javascript"
  style={shadesOfPurple}
  className="syntaxhighlighter"
>
  const countries = ["USA", "Japan", "France", "Italy", "Japan", "Russia",
  "Italy"]; const uniqueCountries = Array.from(new Set(countries)); // returns
  ["USA", "Japan", "France", "Italy", "Russia"]
</SyntaxHighlighter>
