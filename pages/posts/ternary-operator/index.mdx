Ternary operator is a substitute of the if/else statement.

<SyntaxHighlighter
  language="javascript"
  style={shadesOfPurple}
  className="syntaxhighlighter font-sans rounded-md"
>
  {`let movie;
if (horrorMovie) {
  movie = "Rec";
} else {
  movie = "Spiderman";
}`}
</SyntaxHighlighter>

Can be shortened like this :
<SyntaxHighlighter
  language="javascript"
  style={shadesOfPurple}
  className="syntaxhighlighter font-sans rounded-md"
>
  {`// condition ? exprIfTrue : exprIfFalse
const movie = horrorMovie ? "Rec" : "Spiderman";`}
</SyntaxHighlighter>
