Ternary operator is a substitute of the if/else statement.

<Code>
  {`let movie;
if (horrorMovie) {
  movie = "Rec";
} else {
  movie = "Spiderman";
}`}
</Code>

Can be shortened like this :

<Code>
  {`// condition ? exprIfTrue : exprIfFalse
const movie = horrorMovie ? "Rec" : "Spiderman";`}
</Code>
