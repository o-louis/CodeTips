[Ternary operator](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Conditional_Operator) is a substitute of the if/else statement. This is the only operator that use three operands.

**`<condition> ? <value if true> : <value if false>`**

If/Else :

<Code>
  {`let movie;
if (horrorMovie) {
  movie = "Rec";
} else {
  movie = "Spiderman";
}`}
</Code>

Can be shortened like this :

<Code>{`const movie = horrorMovie ? "Rec" : "Spiderman";`}</Code>
